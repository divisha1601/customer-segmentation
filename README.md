# Customer Segmentation using K-Means Clustering

## Project Overview
This project uses K-Means clustering to group customers based on income and spending behavior.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Steps
1. Data Loading
2. Data Visualization
3. Elbow Method
4. K-Means Clustering
5. Cluster Visualization

## Key Insights
- High income, high spending → Premium customers
- High income, low spending → Potential customers
- Low income, high spending → Impulsive buyers
- Low income, low spending → Budget customers

## Conclusion
Customer segmentation helps businesses target customers effectively.
